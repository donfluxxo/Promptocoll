# Promptocoll 📝🤖  
KI - Logbuch: Ein kleines Logbuch zur Dokumentation von KI-Nutzung im Studium

Promptocoll ist eine kleine Desktop-App, die ich gebaut habe, um meine **KI-Nutzung während der Bachelorarbeit sauber zu dokumentieren**.

Viele Hochschulen verlangen inzwischen, dass man transparent angibt,
- **wo** KI genutzt wurde,
- **wofür** sie eingesetzt wurde
- und **wie** die eigene Leistung dabei aussah.

Genau dafür ist Promptocoll gedacht.

---

## 💡 Wofür ist das Tool?

Mit Promptocoll kann man jede KI-Interaktion als Eintrag festhalten:
- verwendeter **Prompt**
- **Antwort** der KI
- optionales **KI-Modell**
- **Datum & Uhrzeit**
- zusätzliche Infos wie:
  - Zweck / Task
  - Kapitel
  - Projekt
  - Tags

Am Ende entsteht ein **übersichtliches, durchsuchbares Logbuch**, das sich z. B. als Grundlage für eine Dokumentation oder einen Anhang in der Arbeit eignet.

---

## ✨ Funktionen (kurz & knapp)

- 📝 Einträge mit Prompt & Antwort erfassen  
- ⏱️ Automatisches oder manuelles Datum  
- 🏷️ Tags, Kapitel, Projekt & Zweck optional  
- 🔍 Suche & Filter im Log  
- 📤 Export als **CSV** oder **Markdown**  
- 💾 Alles lokal gespeichert (JSON-Datei)  
- 🔒 Kein Internet, kein Tracking, keine Cloud
